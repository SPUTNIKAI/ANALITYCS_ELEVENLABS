# Active Context

Текущий фокус:
- Развёртывание сервиса на Render.com (сервер+PostgreSQL)
- Финализация пайплайна LLM (полные поля, `summary_ru`) и запись в БД
- Подготовка включения интеграции БПМсофт в проде (после UAT)

Недавние изменения:
- Добавлены DEBUG‑логи по пайплайну, ngrok‑инструкции
- Реализована интеграция отправки в БПМсофт (фича‑флаг, best‑effort)
- Обновлён промпт LLM: теперь возвращает 8 полей, `summary_ru` вместо `summary`
- Исправлена конфигурация SSL для локальной БД; добавлен compliance‑документ
 - Добавлена идемпотентность отправок в CRM (`crm_dispatches`), dry‑run и заголовок `Origin` по доверенному домену; админ‑ресенд `/api/events/:id/resend-crm`

Следующие шаги:

- Реализовать админ‑роуты (список/детали/репроцессинг/health)

Активные решения/риски:
